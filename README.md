# Scala Cheat Sheet

Scalaのチートシートです.

## Dependencies

* Sbt
* Node.js 24系

## 起動

### Scala.js

```
sbt ~fastLinkJS
```

cf. [Scala.js](https://www.scala-js.org/)

### Vite

```
# yarnが無ければ取得
npm -g install yarn

# nodeモジュール取得
yarn

# 実行
yarn dev
```

cf. [Vite](https://ja.vite.dev/)
